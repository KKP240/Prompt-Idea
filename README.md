## Installation & Setup
**สิ่งที่ต้องมีก่อนเริ่ม:**
1. Vs Code
2. Node.js

### การติดตั้ง
1. Clone โปรเจคลงเครื่อง Local

```bash
git clone https://github.com/KKP240/Prompt-Idea.git

```

2. ติดตั้ง Dependencies ลงในโปรเจคทั้งหมด

```bash
npm install

```

3. Start Development Server

```bash
npm run dev

```
